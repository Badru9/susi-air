# Susi Air - Pilot App

## Setup

```bash

#npm
npm install

#yarn
yarn install

#pnpm
pnpm install

#bun
bun install

```

## Run

```bash
#npm
npm run dev

#yarn
yarn dev

#pnpm
pnpm dev

#bun
bun dev
```

## Build

```bash
#npm
npm run build

#yarn
yarn build

#pnpm
pnpm build

#bun
bun build
```

## Libraries & Rationale

- **Nuxt**: Framework for SSR/SEO.
- **Vue Router**: Navigation.
- **Pinia**: State management.
- **Chart.js / vue-chartjs**: Data visualization.
- **date-fns**: Date manipulation.
- **lucide-vue-next**: Icons.
